Kev's User Guide

1. Introduction

Kev's Textractor is a tool that extracts textures from images. It allows you to select a specific area of an image, adjust its aspect ratio, and save it as a new texture file.

2. Loading an Image

  • Click the "Load Image" button or use the shortcut Ctrl+O.
  • Select an image file from your computer. Supported formats include PNG, JPEG, TIFF, and BMP.
  • The image will appear in the main canvas on the left side of the application.

3. Selecting the Texture Area

  • Click on four points on the image to define the area you want to extract.
  • The points should be selected clockwise, starting from the top-left corner of your desired texture.
  • A blue outline will appear, connecting your selected points.
Tip: You can adjust the points by clicking and dragging them after initial placement.

4. Setting the Aspect Ratio

  • Use the "Aspect Ratio" dropdown to choose between:
    • Estimated: Automatically calculates based on your selection
    • Square: Forces a 1:1 aspect ratio
    • Custom: Allows you to enter a specific aspect ratio
  • For custom aspect ratios, enter the desired value (e.g., 16:9 would be entered as 1.78).

5. Previewing the Texture

  • After selecting the area, a preview of the extracted texture will appear in the right panel.
  • Use the checkboxes below the preview to apply transformations:
    • Flip Vertically: Flips the texture upside down
    • Flop Horizontally: Mirrors the texture left to right
    • Rotate 90Â° Clockwise: Rotates the texture 90 degrees

6. Saving the Texture

  • Click the "Save Texture" button or use the shortcut Ctrl+S.
  • Choose a location and file name for your texture.
  • Select the desired file format (PNG, JPEG, TIFF, or BMP).
Tip: PNG is recommended for textures that require transparency or lossless quality.

7. Additional Features

  • Undo/Redo: Use Ctrl+Z to undo and Ctrl+Y to redo actions.
  • Clear Selection: Removes all selected points and clears the preview.
  • Recent Files: Access recently opened files from the File menu.

8. Tips for Best Results

  • Ensure your source image is well-lit and in focus for the best texture quality.
  • When selecting points, try to align them with visible edges or corners in the image for accurate perspective correction.
  • Experiment with different aspect ratios to find the best fit for your texture needs.

9. Troubleshooting

  • If the application becomes unresponsive, try clearing your selection and reloading the image.
  • Ensure you have the latest version of Kev's Textractor installed for the best performance and feature set.
  • If you encounter any errors, check the application logs for more detailed information.
